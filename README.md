# Bridgeland County Prison - Top-Down Simulation

A polished, realistic top-down prison simulation prototype featuring character creation, NPC behavior, daily operations, and modern minimalist UI.

## Project Structure

- `/Assets/` - Game assets and resources
- `/Scripts/` - Core gameplay systems
- `/Scenes/` - Game scenes and levels
- `/UI/` - User interface components
- `/Audio/` - Sound and ambient audio

## Core Features

### Map & Environment
- Realistic prison layout with progressive security levels
- Multiple buildings: Administration, Medical, Kitchen, Programs, Housing Units
- Gates and secured doors with keycard access
- Recreation yard and control center

### Character System
- Role-based character creation (Officer/Inmate)
- Face generation with eyes, expressions, and personality
- Dynamic eye tracking and blinking
- Emotional state visualization

### NPC Behavior
- Daily schedule automation (6:00 AM - 10:00 PM)
- Intelligent patrol routes and task assignment
- Natural movement and idle animations
- Conversation system

### Security Systems
- 6-level access control system
- Keycard-based door/gate access
- Automatic door/gate mechanics
- Access denial feedback

### User Interface
- Minimalist, modern design inspired by Apple
- Real-time HUD with location and time
- Character, inventory, map, and schedule menus
- Context-sensitive interaction prompts

## Design Philosophy

**Visual Style:** Modern, realistic, professional, clean
**Color Palette:** Concrete gray, asphalt black, chain-link silver, grass green
**Camera:** Strict top-down, smooth follow, no rotation/zoom
**Audio:** Ambient environmental sounds, access/denial beeps
